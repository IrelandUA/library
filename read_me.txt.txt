In the folder "front".

1) Set dependencies - "npm i"

2) Start the frontend of the project - "npm run dev"

3) Open in browser - "http://localhost:5173"




In the folder "server".

1) Set dependencies - "npm i"

2) Plug your database into a file "./app_api/models/db.js"
   Set in "const dbURI" the address of the connection to your database.
	
3) Start the server of the project - "node index.js".
   The server will start on port - "2800".
   You can open it in your browser - "http://localhost:2800".

4) The server may ask you to install - "npx playwright".
   Install "playwright" according to the instructions in the terminal. 
   Start the server of the project - "node index.js".
